Υλικά

1 Arduino

1 τροφοδοτικό

1 LED

1 αντίσταση 220 Ω

1 breadboard

2 καλώδια Male-Male

Συνδεσμολογία

Θετικό ποδαράκι του LED –> Arduino D13

Αρνητικό ποδαράκι του LED –> 1ο άκρο αντίστασης

2ο άκρο αντίστασης -> Arduino GND



Κώδικας

int led = 13;

void setup() {

  pinMode(led, OUTPUT);

}



void loop() {

  digitalWrite(led, HIGH);

  delay(1000);

  digitalWrite(led, LOW);

  delay(1000);

}
