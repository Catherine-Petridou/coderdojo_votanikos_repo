Συνδεσμολογία

Driver PWMA ή ENA --> Raspberry Pi 21

Driver AIN1 ή INA1 --> Raspberry Pi 26

Driver ΑΙΝ2 ή INA2 --> Raspberry Pi 19

Driver VM --> Raspberry 5V

Driver VCC --> Raspberry 5V

Driver GND --> Raspberry GND

Driver A1 --> πρώτη άκρη του κινητήρα

Driver A2 --> δεύτερη άκρη του κινητήρα





Scripting mode

from gpiozero import PWMOutputDevice

from gpiozero import DigitalOutputDevice

from time import sleep

PWM_DRIVE = 21

FORWARD = 26

REVERSE = 19

driveLeft = PWMOutputDevice(PWM_DRIVE, True, 0, 1000)

forwardLeft = DigitalOutputDevice(FORWARD)

reverseLeft = DigitalOutputDevice(REVERSE)



def Stop():
     forwardLeft.value = False
     reverseLeft.value = False
     driveLeft.value = 0


def Forward():
     forwardLeft.value = True
     reverseLeft.value = False
     driveLeft.value = 1.0


def Backward():
     forwardLeft.value = False
     reverseLeft.value = True
     driveLeft.value = 1.0


Stop()

Forward()

sleep(1)

Backword()
