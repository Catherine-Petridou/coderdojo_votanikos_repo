Οδηγίες

Σε αυτή την κατασκευή δεν έχουμε python εντολές.

Δίνουμε τις εντολές στο περιβάλλον που βρισκόμαστε μόλις μπούμε στο Raspberry Pi.

Χωρίς να πατήσουμε python ή nano.

Αυτό το είδος εντολών το ονομάζουμε bash ή shell



Υλικά

1 Raspberry Pi

1 τροφοδοτικό

1 Ηχεία



Συνδεσμολογία

Το καλώδιο των ηχείων στην υποδοχή ήχου του Raspberry Pi



bash εντολές

ls

Θα μας βγάλει μία λίστα με τα αρχεία που έχει ο φάκελος που είμαστε.

Αν, για παράδειγμα, έχουμε ένα mp3 αρχείο με το όνομα a.mp3) τότε δίνουμε

mpg321 a.mp3

Για να το διακόψουμε πατάμε Ctrl+C

Αν θέλουμε να αλλάξουμε την ένταση του ήχου (πχ στο 80%) γράφουμε

amixer cset numid=1 80%



Βοηθητικές εντολές

pwd

για να δούμε σε ποιον φάκελο βρισκόμαστε

cd /a/b

Για να πάμε στον φάκελο /a/b
