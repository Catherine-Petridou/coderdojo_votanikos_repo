Οδηγίες

Το SenseHat θα το τοποθετήσει πάνω στο Raspberry Pi ένας mentor.



Υλικά

1 Raspberry Pi

1 τροφοδοτικό

1 SenseHat



Interactive mode

from sense_hat import SenseHat

sense = SenseHat()

sense.show_message("kalimera")

sense.show_message("kalimera", scroll_speed=0.05)

sense.show_letter("a")

sense.clear()

sense.show_message("kalimera", text_colour=(255, 0, 0))

sense.show_message("kalimera", text_colour=(255, 0, 0), back_colour=(0, 0, 255))



temp = sense.get_temperature()
sense.show_message(temp)


Η τελευταία εντολή θα αποτύχει. Δώσε τις παρακάτω δύο εντολές και μετά ξανά τη show_message. Καταλαβαίνεις τι κάνουν αυτές οι εντολές; temp = round(temp, 2)
temp = str(temp)
