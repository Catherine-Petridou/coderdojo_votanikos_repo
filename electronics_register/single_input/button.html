Συνδεσμολογία

Ένα άκρο του κουμπιού -> GPIO 2

Άλλο άκρο του κουμπιού -> Γη



Script mode (πρώτος τρόπος)

from gpiozero import Button

button = Button(2)

while True:

     if button.is_pressed:

         print("Το κουμπί είναι πατημένο")

     else:

         print("Το κουμπί δεν είναι πατημένο")



Script mode (δεύτερος τρόπος)

from gpiozero import Button

button = Button(2)

button.wait_for_press()

print("Το κουμπί είναι πατημένο")
