Κόκκινο του αισθητήρα -> 5 V

Κίτρινο του αισθητήρα -> GPIO 4

Μαύρο του αισθητήρα -> Γη

Μακρύ ποδαράκι του LED -> GPIO 16

Κοντό ποδαράκι του LED -> Γη



Script mode

from gpiozero import MotionSensor, LED

pir = MotionSensor(4)

led = LED(16)

while True:

     if pir.is_active:

         led.on()

     else:

         led.off()
